# ilm_books_backend

## 1. Create Virtual Environment

Create venv
  
```python

python3 -m venv ilm_books_venv

```

Activate venv
  
```python

source ilm_books_venv/bin/activate

```

## 2. Install Django
  
```python

pip install django
```
